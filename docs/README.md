# Documentation Directory

Comprehensive documentation for Engine Ops platform.

## Structure

- **`api/`** - Auto-generated API documentation (TypeDoc)
- **`context-engineering/`** - Context Engineering System docs
  - `README.md` - Overview
  - `AAR_TEMPLATE.md` - After Action Review template
  - `SECURITY_PRAXIS.md` - Security best practices

- **`prompt-engineering/`** - Prompt Engineering System docs
  - `PROMPT_GOVERNANCE.md` - Prompt versioning and management
  - `SAFETY_PRAXIS.md` - AI safety guidelines
  - `TOKEN_ECONOMICS.md` - Cost optimization strategies
  - `EVAL_TEMPLATE.md` - Evaluation framework

- **`playbooks/`** - Operational runbooks
  - `ci-failure-response.md` - CI debugging guide
  - `deployment-procedures.md` - Blue-Green deployment steps

- **`planning/`** - Historical planning documents
  - Implementation plans, roadmaps, analysis

- **`reports/`** - Project reports
  - `project_manager_summary.md` - Executive summary
  - `engineering_deep_dive.md` - Technical deep dive

## Usage

- **For Operators:** Start with `playbooks/`
- **For AI Engineers:** See `prompt-engineering/`
- **For SREs:** See `context-engineering/`
- **For PMs:** Check `reports/`
