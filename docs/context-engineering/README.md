## üîÑ The Context Cycle

1.  **Observe:** Use Computational Context (Logs/Metrics) to understand system state.
2.  **Orient:** Use SITREPs to align team understanding.
3.  **Decide:** Use Governance models to make decisions.
4.  **Act:** Use PRs and Issue tracking to execute changes.
5.  **Analyze:** Use AARs to learn from outcomes and update context.
