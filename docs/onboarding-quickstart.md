# Engine-Ops Onboarding Quickstart

> **Version:** v1.0  
> **Last updated:** 2025-11-24

---

This guide helps both technical and non-technical users get started with Engine-Ops quickly and confidently.

## For Technical Users

1. **Clone the Repository**

   ```sh
   git clone https://github.com/hummbl-dev/engine-ops.git
   cd engine-ops
   ```

2. **Install Dependencies**

   ```sh
   npm install
   pytest agentic_workflow/tests/
   ```

3. **Review Documentation**
   - [Technical Overview](docs/engine-ops-overview-technical.md)
   - [Copilot Instructions](.github/copilot-instructions.md)

4. **Deploy Locally or to Cloud**
   - See `infra/README.md` for Helm, Kubernetes, and Terraform setup.

## For Non-Technical Users

1. **Managed Service Option**
   - Contact support for white-glove onboarding and managed deployment.

2. **No-Code Automation**
   - Use visual workflow builders and dashboards (coming soon).

3. **Explore Documentation**
   - [Non-Technical Overview](docs/engine-ops-overview-non-technical.md)
   - FAQ and product guides in `docs/`

## Support & Feedback

- For help, open an issue or contact support.
- Feedback is welcome to improve onboarding and documentation.
